Pour le bon fonctionnement du site Hamamet, il faut tout d'abord exécuter le fichier "scriptHamamet.sql" qui se trouve dans le dossier "db" du projet. La base de données s'appelle "Hamamet". la commande pour l'exécuter est : "sudo mysql -u root -p < scriptHamamet.sql" car le login est "root" et il n'y a pas de mot de passe (réappuyer sur Entrée).

Ensuite le chemin du projet est "http://localhost/HamametV2/public/". Nous essayerons de le déployer sur un serveur si on peut le faire gratuitement, je vous enverrai un mail de confirmation de toute façcon.

Une fois sur la page d'accueil, on peuut décider soit de visualiser les produits ou de s'enregistrer. Si l'on veut ajouter des produits au panier sans être enregistrer, on est redirigé vers la page de connexion/inscription.
Lorsqu'un admin est connecté, un bouton apparait sur la navbar lui permettant de visualiser les commandes de tous les utilisateurs.
